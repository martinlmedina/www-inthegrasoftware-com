Using the existing website context of Inthegra Software, implement the HealthCare product page mockup as a premium enterprise landing page.

GENERAL OBJECTIVE
Create a high-end, conversion-oriented, enterprise-grade product page for Inthegra HealthCare.

The page must position Inthegra HealthCare as:
- a robust healthcare software platform
- modular and configurable
- enterprise-ready
- Oracle-native
- backed by 23 years of industry know-how

This is NOT generic software.
This is a serious healthcare platform for complex organizations.

TARGET ORGANIZATIONS
The page must clearly support and communicate coverage for these healthcare business models:
- Prepagas
- Obras sociales
- Gerenciadores de salud
- Clínicas
- Emergencias médicas
- Medicina Laboral
- Internación domiciliaria

IMPORTANT STRATEGIC MESSAGE
Inthegra HealthCare should be perceived as:
- a configurable modular healthcare platform
- built for real operational complexity
- capable of scaling
- designed for integration
- credible for enterprise healthcare environments in LATAM

The page should feel premium, corporate, modern, high-trust, and visually clean.

VISUAL / DESIGN DIRECTION
The visual tone must be:
- premium
- enterprise
- healthcare-specific but not cliché
- modern and highly readable
- conversion-oriented
- Oracle-native in perception

Avoid:
- startup style
- playful SaaS style
- generic healthcare stock-page feeling
- feature overload
- overuse of bright medical clichés
- anything that looks cheap or template-like

IMPORTANT BACKGROUND GRAPHICS GUIDELINE
Inthegra already has graphic assets for backgrounds:
- Solutions
- Product
- Inthegra
- Services

These graphics should be incorporated subtly across the page as delicate background elements.

Rules for using these graphics:
- they must be soft and secondary
- they must not compete with the content
- they must not reduce readability
- they should be used as premium visual texture, not as decoration overload
- use low opacity, large scale, partial cropping, soft blur if needed
- prioritize clean composition over visibility of the asset itself
- they should help reinforce brand sophistication and page depth

Recommended usage:
- hero background: one soft branded graphic shape, partially visible
- section transitions: very subtle graphic accents
- CTA area: controlled branded background detail
- avoid repeating the same pattern too aggressively

PAGE GOAL
The page should help a visitor quickly understand:
1. Inthegra has real healthcare expertise
2. the platform is modular and adaptable
3. Inthegra understands the complexity of different healthcare organizations
4. this is a serious enterprise solution, not generic software
5. the next action should be to schedule a demo
